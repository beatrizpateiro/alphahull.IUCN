## Overview

Computation of the the alphahull according to the IUCN Red List Categories and Criteria

## Installation

Get the latest version from GitHub:

``` r
# Install the package
library(devtools)
install_github("beatrizpateiro/alphahull.IUCN")

# Load package
library(alphahull.IUCN)
```
